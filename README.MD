# Celeste

Celeste is a personal project focused on automation, artificial intelligence, and helping users organize their notes, finances, tasks, and professional activities.

The main programming language used in the project is Python, chosen for its familiarity and because it is well-suited to the planned features. For the development of the web application, I am using the Django framework.

Initially, the goal is to build a simple, fast, and functional application geared toward testing and validating ideas. Later, more advanced features will be implemented, allowing the system to adapt to each user’s needs and routine.

## Technologies Used

- Python 3.12.3
- Django 6.0.6
- PostgreSQL
- Docker
