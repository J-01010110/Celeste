# Celeste

Celeste é um projeto pessoal focado em automação, inteligência artificial e apoio ao usuário na organização de anotações, finanças, tarefas e atividades profissionais.

A principal linguagem de programação utilizada no projeto é Python, escolhida por familiaridade e por ser adequada às funcionalidades planejadas. Para o desenvolvimento da aplicação web, estou utilizando o framework Django.

Inicialmente, o objetivo é construir uma aplicação simples, rápida e funcional, voltada para testes e validação das ideias. Posteriormente, serão implementadas funcionalidades mais avançadas, permitindo que o sistema se adapte às necessidades e à rotina de cada usuário.

## Tecnologias utilizadas

- Python 3.12.3
- Django 6.0.6
- PostgreSQL
- Docker
